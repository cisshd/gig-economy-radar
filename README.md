# 📡 رادار القوى العاملة الرقمية
## Saudi Gig Economy Radar — هاكاثون الابتكار في البيانات 2026

### 🎯 فكرة المشروع
مرصد ذكي يقدّر حجم وتوزيع العمالة في اقتصاد المنصات بالمملكة العربية السعودية  
عبر تحليل الإشارات الرقمية العامة (Google Trends + App Store Reviews)،  
مع مقارنة النتائج بالأرقام الرسمية لإظهار "الفجوة الإحصائية".

---

### 🚀 طريقة التشغيل

#### 1. تثبيت المتطلبات
```bash
pip install -r requirements.txt
```

#### 2. تشغيل جمع البيانات
```bash
python data_collector.py
```

#### 3. تشغيل لوحة البيانات
```bash
streamlit run dashboard.py
```

---

### 🧮 المنهجية
```
تقييمات التطبيق الشهرية
        ÷ 2% (معدل التقييم)
        = عدد الرحلات اليومية
        ÷ 7 (متوسط رحلات/سائق/يوم)
        × معامل Google Trends
        = تقدير عدد العمال النشطين
```
